## 📌 사용 가이드

**vscode 환경에서 수행해 주세요.**
   

>가상환경 생성

```bash
python -m venv 가상환경이름
```
>가상환경 실행
```bash
가상환경이름/Scripts/activate.bat
```
>패키지 설치
```bash
pip install -r requirements.txt
```
>실행
```bash
python manage.py runserver
```
